## How to use
- Unzip and open in VS Code.
- Edit `index.html` to update content or add profile image (`.avatar-placeholder`).
- Push to GitHub and deploy to Render or GitHub Pages.

## Push to GitHub (example)
```bash
git init
git add .
git commit -m "Initial portfolio v2"
git branch -M main
git remote add origin https://github.com/Abbas5055/abbas-portfolio.git
git push -u origin main
```
